# Sweet Shop Management System

This is a simple full-stack Sweet Shop Management System project.  
I made this project to understand how frontend and backend work together in a real application.

The project is structured in a single repository with separate folders for frontend and backend.

---

## Project Structure
```
sweet-shop-management-system
├── sweet-shop-backend
├── sweet-shop-frontend
└── .gitignore
```

---

## Tech Stack Used

### Frontend
- React (v19+) 
- React DOM  
- React Scripts (CRA) 
- Testing Library  
- Web Vitals  
- CSS / Material-UI  

### Backend
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pytest

---

## How to Run the Project

### Backend Setup

```bash
cd sweet-shop-backend
```
Install dependencies:

```
pip install -r requirements.txt
```
Run backend server:
```
python app.py

```


Backend runs on:
```
http://localhost:5000
```
Frontend Setup
```
cd sweet-shop-frontend

```
Install node modules:
```
npm install
```
Start frontend:

```bash
npm start
```

Frontend runs on:
```arduino

http://localhost:3000
```
