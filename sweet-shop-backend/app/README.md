# Sweet Shop Management System (Backend)

This project is a backend API for managing a Sweet Shop.  
It allows users to register, log in, and manage sweets inventory including adding and viewing sweets.

The project is built as part of an assignment to demonstrate backend development, basic authentication, database integration, testing, and version control practices.
